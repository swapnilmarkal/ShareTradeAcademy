# 📈 Share Trade Academy

A professional stock trading education and mentorship website designed to help beginners and aspiring traders learn market fundamentals, technical analysis, and disciplined trading strategies.


## 🌐 Live Preview
https://www.sharetradeacademy.in/


## 📌 Project Overview

Share Trade Academy is a responsive, modern landing page for a trading education institute.  
It showcases trading programs, mentor details, student testimonials, FAQs, and a contact system for lead generation.

The goal of this project is to create a **high-conversion educational landing page** with a premium UI/UX feel.


## 🚀 Features

- ⚡ Fully responsive design (mobile, tablet, desktop)
- 🎯 Conversion-focused landing structure
- 📊 Trading programs section
- 👨‍🏫 Mentor profile section
- 💬 Student testimonials slider (horizontal scroll)
- ❓ FAQ accordion system
- 📩 Contact form integration
- 📍 Google Maps embed
- 📈 Live market ticker widget (TradingView)
- 🎨 Modern dark UI with glow effects


## 🛠️ Tech Stack

- HTML5
- CSS3 (Custom variables + Flexbox + Grid)
- Vanilla JavaScript
- TradingView Widget (Market ticker)
- Google Maps Embed API
- Lucide Icons
